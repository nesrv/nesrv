Шпаргалка по Markdown:

### Заголовки
```markdown
# H1
## H2
### H3
#### H4
##### H5
###### H6
```

### Текст
```markdown
*курсив* или _курсив_
**жирный** или __жирный__
**_жирный курсив_**
~~зачёркнутый~~
`код`
```

### Списки
```markdown
- Неупорядоченный
* Список
+ Варианты

1. Упорядоченный
2. Список
```
